# Actor Metadata
title: YouTube Scraper
description: Scrapes videos, channels, playlists, and hashtags from YouTube. Acts as an unlimited alternative to the official API. Supports filters, subtitle downloads, and multiple input modes (search or direct URLs).
version: 0.0
build_id: 9Gpb8NaEA6ySAfUte

## Parameters

### searchQueries
type: array[string]
description: Search terms or keywords, same as typing them into YouTube's search bar.
behavior: Each query triggers a YouTube search. Results can be filtered and sorted using parameters like `sortingOrder`, `dateFilter`, and `lengthFilter`.
example: ["ai podcast", "agentic rag", "n8n automation"]

### maxResults
type: integer
minimum: 0
maximum: 999999
default: 0
description: Maximum number of regular videos to scrape per query or channel.
behavior: Acts as a cap on total results fetched for each search or channel. Does not include Shorts or Streams.
example: 100

### maxResultsShorts
type: integer
minimum: 0
maximum: 999999
default: 0
description: Maximum number of Shorts to scrape per query or channel.
behavior: Controls the number of short-form videos collected.
example: 25

### maxResultStreams
type: integer
minimum: 0
maximum: 999999
default: 0
description: Maximum number of live or past streams to scrape per query or channel.
behavior: Limits the count of stream results per source.
example: 10

### startUrls
type: array[object]
description: Direct URLs to scrape (videos, channels, playlists, hashtags, or search result pages).
behavior: Overrides `searchQueries`. Each URL must be provided as an object with a `url` key. Supports multiple sources in one run.
example:
[
  {"url": "https://www.youtube.com/c/Apify"},
  {"url": "https://www.youtube.com/hashtag/ai"},
  {"url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ"}
]

### downloadSubtitles
type: boolean
description: Enables subtitle downloads for scraped videos.
behavior: When true, subtitles are downloaded and converted to the selected `subtitlesFormat` (requires `subtitlesLanguage`).
example: true

### saveSubsToKVS
type: boolean
description: Saves downloaded subtitles to a Key-Value store.
behavior: Only works if `downloadSubtitles` is true. Stores subtitles in the specified format (e.g., `.srt`, `.vtt`).
example: true

### subtitlesLanguage
type: string
enum: ["any", "en", "de", "es", "fr", "it", "ja", "ko", "nl", "pt", "ru"]
default: "en"
description: Language code for subtitle downloads.
behavior: Defines which subtitles to retrieve. Requires `downloadSubtitles` enabled.
example: "en"

### preferAutoGeneratedSubtitles
type: boolean
description: Prefers auto-generated subtitles when available.
behavior: Works with `subtitlesLanguage`. Prioritizes automatically generated captions over manually uploaded ones.
example: true

### subtitlesFormat
type: string
enum: ["srt", "vtt", "xml", "plaintext"]
default: "srt"
description: File format for downloaded subtitles.
behavior: Determines subtitle file type and storage format.
example: "vtt"

### sortingOrder
type: string
enum: ["relevance", "rating", "date", "views"]
description: Defines YouTube sorting order for search results.
behavior:
- `relevance`: Default behavior, matches YouTube search.
- `rating`: Sorts by video rating.
- `date`: Newest videos first.
- `views`: Sorts by view count.
example: "date"

### dateFilter
type: string
enum: ["hour", "today", "week", "month", "year"]
description: Filters search results by upload timeframe.
behavior: Restricts results to a specific upload window. Works only for search-based scraping.
example: "week"

### videoType
type: string
enum: ["video", "movie"]
description: Filters for specific YouTube video types.
behavior: `video` returns standard uploads; `movie` returns films or rentals.
example: "video"

### lengthFilter
type: string
enum: ["under4", "between420", "plus20"]
description: Filters videos by duration.
behavior:
- `under4`: Under 4 minutes.
- `between420`: 4–20 minutes.
- `plus20`: Over 20 minutes.
example: "between420"

### isHD
type: boolean
description: Restricts to HD videos.
behavior: Filters results to include only videos available in HD.
example: false

### hasSubtitles
type: boolean
description: Restricts to videos with subtitles or closed captions.
behavior: Filters only videos that contain captions.
example: true

### hasCC
type: boolean
description: Restricts to Creative Commons licensed videos.
behavior: Filters for videos under Creative Commons license.
example: false

### is3D
type: boolean
description: Restricts to 3D videos.
behavior: Returns 3D-capable videos only.
example: false

### isLive
type: boolean
description: Restricts to live streams or upcoming live videos.
behavior: Includes live or scheduled streams only.
example: false

### isBought
type: boolean
description: Restricts to purchased or premium videos.
behavior: Includes only videos available via YouTube purchase or subscription.
example: false

### is4K
type: boolean
description: Restricts to 4K videos.
behavior: Filters results to 4K resolution content only.
example: false

### is360
type: boolean
description: Restricts to 360° videos.
behavior: Returns immersive spherical videos only.
example: false

### hasLocation
type: boolean
description: Filters for videos with geolocation metadata.
behavior: Only includes videos tagged with a physical location.
example: false

### isHDR
type: boolean
description: Restricts to HDR videos.
behavior: Returns only videos that support high dynamic range.
example: false

### isVR180
type: boolean
description: Restricts to VR180 format videos.
behavior: Returns stereoscopic VR180 content.
example: false

### oldestPostDate
type: string
pattern: "^(\\d{4})-(0[1-9]|1[0-2])-(0[1-9]|[12]\\d|3[01])(T[0-2]\\d:[0-5]\\d(:[0-5]\\d)?(\\.\\d+)?Z?)?$|^(\\d+)\\s*(minute|hour|day|week|month|year)s?$"
description: Minimum video publish date. Accepts absolute (YYYY-MM-DD) or relative ("3 days") values.
behavior: Filters to videos newer than this date. Automatically resets sorting to `NEWEST` when used.
example: "2 weeks"

### sortVideosBy
type: string
enum: ["NEWEST", "POPULAR", "OLDEST"]
description: Defines sorting when scraping a specific channel or playlist.
behavior:
- `NEWEST`: Fetches newest uploads first.
- `POPULAR`: Fetches most popular videos first.
- `OLDEST`: Fetches oldest uploads first.
example: "POPULAR"

## Example Input
```json
{
  "searchQueries": ["ai podcast", "n8n workflow"],
  "maxResults": 50,
  "maxResultsShorts": 10,
  "maxResultStreams": 5,
  "startUrls": [
    {"url": "https://www.youtube.com/c/Apify"},
    {"url": "https://www.youtube.com/hashtag/ai"}
  ],
  "downloadSubtitles": true,
  "saveSubsToKVS": true,
  "subtitlesLanguage": "en",
  "preferAutoGeneratedSubtitles": true,
  "subtitlesFormat": "vtt",
  "sortingOrder": "date",
  "dateFilter": "month",
  "videoType": "video",
  "lengthFilter": "plus20",
  "isHD": true,
  "hasSubtitles": true,
  "is4K": false,
  "isLive": false,
  "sortVideosBy": "NEWEST"
}
```
